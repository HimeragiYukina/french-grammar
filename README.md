# 法语语法手册 A1–C2 · Grammaire française

一份单页、离线可用的法语语法手册，覆盖欧标 **A1–C2** 各级别，中法对照，带搜索、级别筛选与深色模式。

🔗 **在线访问：** https://himeragiyukina.github.io/french-grammar/

## 特性

- 📚 按 A1–C2 级别组织的语法条目
- 🔍 实时搜索与级别筛选
- 🌗 浅色 / 深色主题切换
- 📱 响应式布局，适配桌面与移动端
- 🦾 纯静态单文件（`index.html`），无需构建、无依赖

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何服务器或构建步骤。

## 部署

本项目通过 GitHub Pages 从 `main` 分支根目录自动部署。
